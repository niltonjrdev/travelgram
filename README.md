# 🌍 Travelgram — Travel Profile Page

A travel-themed profile page inspired by social media layouts, built with semantic HTML and modular CSS.

## ✨ Features

- 👤 Profile section with avatar, bio, and stats (countries visited, photos count, location)
- 🖼️ Responsive photo gallery grid with 12 travel images
- 🧭 Navigation bar with search icon, links, and profile picture
- 🗂️ Modular CSS architecture — one file per section (header, nav, main, footer, global)
- 🧩 Semantic HTML — `<header>`, `<nav>`, `<main>`, `<footer>`, `<figure>`

## 🛠️ Tech Stack

- HTML5
- CSS3 (modular — footer, global, header, index, main, nav)
