# 🌍 Travelgram — Página de Perfil de Viagens

Página de perfil com tema de viagens inspirada em layouts de redes sociais, construída com HTML semântico e CSS modularizado.

## ✨ Funcionalidades

- 👤 Seção de perfil com avatar, bio e estatísticas (países visitados, total de fotos, localização)
- 🖼️ Galeria de fotos em grid com 12 imagens de viagem
- 🧭 Barra de navegação com ícone de busca, links e foto de perfil
- 🗂️ Arquitetura CSS modular — um arquivo por seção (header, nav, main, footer, global)
- 🧩 HTML semântico — `<header>`, `<nav>`, `<main>`, `<footer>`, `<figure>`

## 🛠️ Tecnologias

- HTML5
- CSS3 (modular — footer, global, header, index, main, nav)
